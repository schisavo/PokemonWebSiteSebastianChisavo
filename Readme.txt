Saludos.
Adjunto proyecto Pokemon web que tiene un login, signup y un forgot password por el correo.
El proyecto esta hecho en HTML,CSS,JAVASCRIPT y PHP.
La base de datos es Mysql.

Para ejecutar el proyecto se requiere Xammp, donde en la carpeta xampp\htdocs ya descomprimido.
Luego se exporta la base de datos adjunta para el registro y logeo.

se coloca esta direccion en el navegador http://localhost/Pokemonweb/index.html
y se ejecuta la base de datos.
